# Expense Tracker
A Full Stack Expense Tracker built using MERN Stack(MongoDB, Express, React, Node.js) that helps user track and manage their income and expenses with dynamic UI and interactive visualizations.

## Features
- User Authentication
- Dynamic User Interface
- Add and Delete income & expense entries
- Integration of chart for better visualization
- Download transaction history in Excel format
  
## Installation and Setup
1. Clone the repository:
```bash
git clone https://github.com/abhisheksth9/MERN-ExpenseTracker.git
cd MERN-ExpenseTracker
```
2. Setup Backend:
```bash
cd backend
npm install
npm run dev
```
3. Setup Frontend:
```bash
cd frontend
npm install
npm run dev
```
